# DigitalWorks/保健室～マジカルピュアレッスン

封包/资源格式处理目录，包含解包、打包或格式转换脚本。

本 README 为目录补充说明，便于后续维护、迁移和复用。

## 文件说明

| 文件 | 说明 |
|---|---|
| `BunBun_Analysis.docx` | 逆向分析/使用说明文档 |
| `font_tool.py` | 字体/字符集处理工具；BunBun Engine font32.dat 字体重绘工具 |
| `pac_tool.py` | 封包解析/解包/重打包工具；BunBun Engine PAC Archive Tool (保健室～マジカルピュアレッスン♪～) |
| `tak_text.py` | 文本提取/回写工具；BunBun Engine TAK Script Tool (保健室～マジカルピュアレッスン♪～) |

## 常见流程

该目录以单文件文本工具为主。建议先查看脚本帮助或源码顶部说明：

```bash
python tak_text.py --help
```

## 注意事项

- 本仓库脚本大多是特定游戏/特定版本适配，跨作品复用前需要重新核对文件头、索引表、指令格式和编码。
- 文本编码通常与原游戏运行时有关，常见为 CP932/SJIS；写入中文前需要确认补丁、Hook、字体或码表映射方案。
- 处理前保留原始文件备份；注入后建议进行二进制比对、游戏内实机检查和异常文本回查。
- 若脚本会重建封包或重排文本区，务必确认 offset、长度字段、压缩块大小和校验字段是否同步更新。
