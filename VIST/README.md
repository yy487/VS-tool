# VIST

资源处理目录，主要覆盖图像、字体或专用素材格式。

本 README 为目录补充说明，便于后续维护、迁移和复用。

## 文件说明

| 文件 | 说明 |
|---|---|
| `emc_tool.py` | 封包解析/解包/重打包工具；EMC (EMSAC-Binary Archive-2) 解包/封包工具 |
| `emi2png.py` | 图像或素材格式处理工具；EMI (EMSAC-Image-2) → PNG 解码器 |
| `EntisGLS_VIST_Analysis.docx` | 逆向分析/使用说明文档 |
| `EntisGLS_VIST_Complete.docx` | 逆向分析/使用说明文档 |
| `eri_fast.c` | C 原生辅助源码 |
| `eri_fast.so` | 加速库或运行时辅助库 |
| `gds_text.py` | 文本提取/回写工具；GDS 脚本文本提取/导入工具 |
| `VIST_傻瓜教程.docx` | 逆向分析/使用说明文档 |

## 常见流程

该目录以单文件文本工具为主。建议先查看脚本帮助或源码顶部说明：

```bash
python gds_text.py --help
```

## 注意事项

- 本仓库脚本大多是特定游戏/特定版本适配，跨作品复用前需要重新核对文件头、索引表、指令格式和编码。
- 文本编码通常与原游戏运行时有关，常见为 CP932/SJIS；写入中文前需要确认补丁、Hook、字体或码表映射方案。
- 处理前保留原始文件备份；注入后建议进行二进制比对、游戏内实机检查和异常文本回查。
